Bienvenidos

Esta es mi primera vez haciendo esto.

Esta página fue creada como parte de un trabajo práctico donde se aplican conocimientos básicos de HTML y CSS. A lo largo del sitio se pueden observar diferentes elementos como encabezados, navegación, secciones, uso de imágenes y estilos personalizados.

Además, se trabajó con conceptos importantes como el uso de colores, tipografías, el modelo de caja (box model) y la organización del contenido mediante alineación.

El objetivo principal de este trabajo es aprender a estructurar una página web y mejorar su diseño utilizando hojas de estilo en CSS.

Diseño con CSS

El uso de CSS permite mejorar la apariencia de una página web, aplicando colores, tipografías y estilos que hacen que el contenido sea más atractivo y fácil de leer para el usuario.

Organización del contenido

Gracias al modelo de caja (box model), es posible organizar los elementos en la página utilizando márgenes, rellenos y bordes, logrando un diseño ordenado y visualmente equilibrado.
